This repository packages Blockstream `electrs` as a standalone Umbrel app for
Liquid mainnet. It depends on Umbrel's Elements app as a Liquid node, exposes
a plain Electrum endpoint on `51001`, and serves a landing page for connecting
Blockstream App and other wallets.
